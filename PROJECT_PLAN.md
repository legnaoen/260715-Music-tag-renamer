# 구현 준비 계획

## 현재 기준

- MVP 대상: Windows, MP3 폴더 선택/드롭(하위 폴더 포함), ID3 미리보기, 선택 적용, 로그
- 유지 규칙: `TIT2`, `TBPM`, `TKEY`; 누락 태그 제외; BPM/Key 접두어 중복 방지
- 안전 기본값: 파일명 변경 없음, 적용 전 선택 목록 표시, 파일별 오류 격리

## 구현 순서

1. Electron 창·보안 IPC·Renderer 골격
2. `music-metadata` 기반 MP3 읽기/쓰기
3. 폴더 선택 및 드래그 앤 드롭
4. 상태·미리보기·선택 적용
5. 로그/진행률/오류 패널
6. 실제 DJ 폴더 회귀 테스트 및 Windows 패키징

## 남은 검증

- 실제 MP3 태그 읽기/쓰기 회귀 테스트
- 백업 파일 생성 및 되돌리기 정책 확정
- Windows 설치 파일과 아이콘 확정
